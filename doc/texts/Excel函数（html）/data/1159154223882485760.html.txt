逻辑函数总文档
一、功能背景及价值
在业务规则制定、流程控制和智能决策等场景中，经常需要对各种条件进行判断和逻辑处理。逻辑函数提供了一套完整的条件判断和逻辑运算工具，能够实现：智能化的业务规则引擎、动态化的流程控制、自动化的决策判断。此时用户便可通过数据上报、流程等模块中的表单设计器直接来为字段配置对应函数公式，也可通过字段联动来设置赋值字段并选择对应函数公式为其赋值。

二、功能概述
本功能包含基础逻辑判断、复合逻辑运算和特殊逻辑处理三大类函数，支持单条件、多条件组合判断以及异常处理，满足不同复杂度的逻辑处理需求。
三、功能使用说明
1. 包含函数介绍
基础逻辑函数
IF：条件判断
TRUE：返回真值
FALSE：返回假值
复合逻辑函数
AND：逻辑与
OR：逻辑或
NOT：逻辑非
IN：包含判断
LIKE：模糊匹配
特殊逻辑函数
ISEMPTY：空值判断
NOTEMPTY：非空判断
IFS：多条件判断
SWITCH：条件选择
SORT：条件排序
IFERROR：异常处理
2. 常用函数介绍
IF 函数
功能：条件判断
语法：IF(条件, 真时返回值, 假时返回值)
示例：IF(成绩>60, "及格", "不及格") → 成绩大于60返回"及格"，否则返回"不及格"
AND 函数
功能：逻辑与运算
语法：AND(条件1, 条件2, ...)
示例：AND(年龄>18, 性别="男") → 年龄大于18且性别为男时返回TRUE
OR 函数
功能：逻辑或运算
语法：OR(条件1, 条件2, ...)
示例：OR(成绩>90, 出勤率>95) → 成绩大于90或出勤率大于95时返回TRUE
IN 函数
功能：包含判断
语法：IN(值, 值1, 值2, ...)
示例：IN(城市, "北京", "上海", "广州") → 城市是北上广之一时返回TRUE
IFS 函数
功能：多条件判断
语法：IFS(条件1, 结果1, 条件2, 结果2, ...)
示例：IFS(成绩>=90, "优秀", 成绩>=80, "良好", 成绩>=60, "及格", TRUE, "不及格")
四、注意点
1.
IFS函数会按顺序判断，第一个满足的条件会返回对应结果
2.
SWITCH函数需要提供默认返回值
3.
空字符串("")和NULL在ISEMPTY判断中都视为空
