一、
功能背景以及价值
各类公文流程的发文拟稿入口，根据流程类型分类展示，用户可以快速发文拟稿。
二、
功能概述
1.
发起公文页面分为公文流程类型结构树和公文流程入口两部分；
2.
公文流程收藏；
3.
支持关键字搜索；
4.
个性化设置；
三、
功能详细说明
（一）前台使用-发起公文页面
功能入口：公文管理-发起公文
发起公文菜单是拟稿公文流程入口，点击左侧菜单的【发起公文】后，在该页面用户看到的都是拥有创建权限的公文流程。
1.
全部流程：
显示所有可以发起的公文流程，显示最常用的部分流程（可自定义是否开启），默认显示类型结构树，可关闭；
点击左侧类型，右侧会自动过滤只显示该类型下的公文流程
2.
我的收藏：
显示个人收藏的流程，可以在流程后点击收藏按钮后收藏此流程，再次点击可以取消收藏。
3.
常用流程：
流程模块-设置-个人设置-功能设置，【提交申请显示常用流程】开关开启时，发起页面-全部流程中会显示常用流程，开关关闭时则不显示。如勾选了【常用流程单独页签显示 】，则常用流程作为单独页签展示。
单独页签展示：
关闭【提交申请显示常用流程】，发起公文页面不展示常用流程
4.
关键字搜索：
发起公文，公文类型结构树和公文流程支持关键字搜索
5.
个性化设置：
发起公文页面右上角支持个性化设置，可以设置左侧流程树结构、视图设置和分类方式：
默认显示类型结构树，可以关闭，关闭后不显示结构树，可点击箭头展开，页面如下：
【默认显示】关闭后可以通过【恢复默认设置】打开【默认显示】，默认展开类型结构树
可以通过【设为默认设置】，将当前设置为默认设置；
点击【同步设置】可以将当前个性化设置同步给其他人，可以设置同步的设置项和同步对象；
视图默认为瀑布视图，可切换为卡片视图
