一、功能背景
在数字化办公环境中，企业数据安全面临诸多挑战，数据泄露风险日益增加。无论是内部文档的不当传播，还是网络页面信息的非法获取，都可能给企业带来严重损失。水印作为一种有效的数据保护手段，能够清晰标记文档和页面的来源，为数据追踪提供关键线索。然而，仅设置水印还不足以满足企业多样化的安全需求，精准控制水印的应用范围同样重要。因此，将水印设置与应用范围管控（即原黑白名单功能）整合为一个功能，能帮助企业更高效地保护数据安全，实现对数据更精细化的管理。
二、功能价值
强化数据安全标识：通过在文档和页面添加水印，明确数据来源，对潜在的数据泄露行为起到威慑作用，降低数据被非法使用和传播的风险。
精准控制水印应用：可根据企业的具体业务需求，灵活设定哪些页面、人员或 IP 范围需要显示水印，哪些不需要，提高水印应用的精准度和灵活性，适应不同场景下的数据安全管理要求。
提升数据追溯效率：精准的水印应用范围控制，使得在数据出现安全问题时，企业能够更快速、准确地追踪数据的传播路径和使用情况，为后续的处理和防范措施提供有力支持。
合规与审计支持：满足企业在数据合规性方面的要求，为内部审计和外部监管提供清晰的数据使用记录，确保企业运营符合相关法规和政策。
三、功能概述
该功能整合了水印设置和应用范围管控两大核心模块。用户可以在同一操作界面中完成水印的创建、编辑以及对水印在不同场景下显示范围的精确控制。水印类型支持文本和图片，并且可以对水印的样式，如不透明度、旋转角度等进行个性化设置。同时，通过设置不同的应用规则，实现对页面、人员和 IP 范围的水印显示与否的管理。
四、业务流程图
登录后台管理中心：使用企业分配的账号和密码登录到后台管理系统，获取操作权限。
进入通用设置：在后台管理中心的菜单中找到 “通用设置” 选项，这是进行水印及应用范围管控功能设置的入口。
点击水印及应用范围管控功能：在 “通用设置” 页面中，选择 “水印设置” 功能模块，进入具体的操作页面。
选择水印类型（文本 / 图片）：在操作页面中，根据需求选择水印类型，系统默认提供文本水印，也可切换为图片水印。
设置水印内容、样式（不透明度、旋转角度等）：针对所选的水印类型，设置具体的内容和样式参数。如文本水印可编辑内容、字体、颜色等；图片水印可上传本地图片，并调整水印的不透明度和旋转角度。
设置应用范围（按页面、人员范围、IP 控制）：在应用范围设置区域，选择按页面、人员范围或 IP 控制水印的显示。对于页面控制，可添加黑白名单页面链接；人员范围控制可选择特定人员、部门等；IP 控制可选择 IP 网段，分别设置其是否显示水印。
保存设置：完成水印和应用范围的所有设置后，点击 “保存” 按钮，使设置生效。
查看操作日志（可选）：在操作页面的右上角，点击 “更多” 按钮，可查看水印及应用范围设置的操作日志，记录了设置时间、操作人及具体操作内容，方便进行追溯和审计。
五、功能详细说明
1、水印设置
进入设置页面：登录后台管理中心，找到 “通用设置”，点击其中的 “水印设置”，进入功能设置页面，包含水印类型选择、内容设置、样式调整以及黑白名单设置等多个区域。
启用 PC 端水印：该开关在初始状态下处于关闭状态。拥有特定权限的用户能够将其开启，一旦开启，PC 端网页便会显示水印。开启此功能后，用户还能够针对水印的内容、样式等方面开展进一步的个性化定制操作。
启用移动端水印：同样默认处于关闭状态，开启之后移动端网页会呈现水印。其设置方式与 PC 端存在一定相似性，但在设置过程中，需要充分考量移动端设备的屏幕尺寸大小、显示特性等因素，以此保证水印在移动端的显示效果优良，不会对页面内容的正常浏览造成干扰。
2、水印类型与内容设置
文本水印：系统默认选择文本水印。水印宽度和高度默认值为 200px，可根据实际需求调整。在内容设置区域，点击右侧提供的变量值，如 “当前操作者姓名”“当前操作者编号”“当前日期” 等，这些变量会自动添加到富文本窗口。用户还可以对添加内容的字体、大小、颜色、格式等进行编辑，以达到满意的显示效果。编辑完成后，点击保存按钮使设置生效。例如，在企业内部财务报表的水印中添加当前日期和财务人员编号，便于数据追溯。
图片水印：若选择图片水印类型，水印内容区域会出现 “上传图片” 图标。点击该图标，系统弹出 “本地文件” 窗口，用户可从本地存储设备中选择合适的图片作为水印。上传时需注意系统对图片格式和大小的要求，避免上传失败。例如，企业可以使用带有公司 logo 的图片作为水印，增强品牌辨识度的同时，实现数据来源标记。
3、水印样式调整
不透明度（百分比）：水印不透明度默认设置为 30%，取值范围为 0 - 100%。用户可根据实际使用场景和需求进行调整，数值越大，水印显示越明显；数值越小，水印越趋近透明。比如在一些展示性的宣传文档中，可将水印不透明度调低，减少对文档美观度的影响，同时仍起到一定的版权保护作用。
旋转角度（逆时针）：旋转角度默认值为 15°，用户可以根据个人喜好和实际需求进行修改。通过调整旋转角度，使水印呈现不同的倾斜效果，满足多样化的设计需求。例如，在一些创意类文档中，调整水印旋转角度可使其与文档风格更契合。
4、水印黑白名单设置
在水印及应用范围管控功能页面中，“应用范围设置” 区域包含了对水印显示范围进行控制的相关选项，是实现精准管控的关键入口。
（1）按页面控制
黑名单设置：选择 “按页面控制” 模式后，点击黑名单的【设置】按钮，进入黑名单列表页面。用户可在此批量添加不需要显示水印的页面链接，每个链接之间用回车分隔。系统具备自动去重功能，防止重复添加。添加完成后，点击 “确定” 按钮，后续访问这些页面时将不会显示水印。黑名单列表中的页面链接支持编辑和删除操作，编辑时会打开单独的编辑页面进行修改；删除操作支持单个删除和批量删除，但都需要二次确认，以防止误操作。例如，企业的公共宣传页面，如产品展示页面，为了保证页面美观度，可将其添加到黑名单，使其不显示水印。
白名单设置：点击白名单的【设置】按钮，进入白名单列表页面。操作方式与黑名单类似，在此添加的页面，在访问时会显示水印。比如企业的机密文件页面，如战略规划文档页面，为了确保数据安全，可将其添加到白名单，保证访问时都带有水印。
（2）按人员范围控制
黑名单设置：点击 “按人员范围控制” 后，再点击黑名单的【设置】按钮，会出现人员选择界面。用户可通过共享组件选择不需要显示水印的人员范围，包括特定的部门、岗位或具体人员等。选择完成后，这些被选中的人员在访问相关页面或文档时将不会看到水印。例如，企业的市场推广部门在对外宣传活动中，为了提高工作效率，可将该部门人员添加到人员黑名单，使其在访问宣传资料页面时不显示水印。
白名单设置：白名单设置操作与黑名单类似，只是在此处设置的人员在访问时会显示水印。比如企业的核心技术团队成员，在访问技术研发相关文档时，为了加强数据安全管理，可将其添加到人员白名单，确保他们看到的文档都带有水印。
（3）按 IP 控制
黑名单设置：点击 “按 IP 控制” 后，再点击黑名单的【设置】按钮，系统会调用已有的网段策略。用户可直接从中选择不需要显示水印的 IP 网段。若系统中没有符合需求的网段，可点击相关链接（需具备相应权限）打开【账号安全】 - 【网段管理】页面进行新建网段操作。例如，企业的访客网络 IP 网段，为了避免水印对访客造成困扰，可将其添加到 IP 网段黑名单。
白名单设置：白名单设置操作与黑名单一致，设置的 IP 网段范围内的用户在访问时会显示水印。比如企业内部办公网络的 IP 网段，为了保障内部数据安全，可将其添加到 IP 网段白名单，确保该网段用户访问相关页面时都能看到水印。
5、操作日志查看
在水印及应用范围管控功能页面的右上角，设有 “更多” 按钮。点击该按钮，用户可以查看操作日志，日志详细记录了设置时间、执行操作的人员以及具体的操作内容，包括水印设置的修改和应用范围的调整等。操作日志为用户追踪设置历史变更提供了便利，在审计工作以及故障排查过程中具有重要作用。例如，当发现水印显示异常或应用范围设置错误时，可通过查看操作日志，快速确定问题发生的时间和操作人，便于及时解决问题。
6、文件水印设置
启用开关：在文件水印设置区域，有一个 “启用” 开关。默认状态为关闭，当您将其切换为开启状态后，文件水印功能将生效。若业务模块未指定文件的水印，将按照当前页面配置展示水印 。
水印类型选择：系统提供多种水印类型供您选择，包括文本、图片、图文、二维码 / 条码、页码。
文本：选择文本水印后，您可以通过点击 “水印内容” 旁边的齿轮图标，在弹出的设置框中输入具体的文本内容，还能对文本的字体、字号、颜色、加粗、倾斜等样式进行编辑。例如，您可以输入公司名称、文件编号等信息作为水印内容。
图片：若选择图片水印，点击 “水印内容” 旁的齿轮图标，可从本地选择合适的图片上传作为水印。需注意图片的格式和大小要符合系统要求，避免上传失败。比如，您可以上传公司的 logo 图片作为文件水印。
图文：此类型允许您将文本和图片结合起来设置水印。点击 “水印内容” 的齿轮图标后，分别设置文本内容和上传图片，并调整它们的相对位置、大小比例等，以达到理想的水印效果。
二维码 / 条码：选择该类型后，点击 “水印内容” 的齿轮图标，可生成或输入特定的二维码或条码信息作为水印。例如，您可以生成包含文件相关信息的二维码，方便扫码获取详细内容。
页码：选择页码水印后，系统会自动在文件上添加页码作为水印内容，您可以设置页码的格式、位置等样式。
应用于选项：“应用于” 选项包括 “查看” 和 “下载” 两个复选框，默认均勾选。这意味着当您启用文件水印后，在查看文件以及下载文件时，都会显示您所设置的水印。您可以根据实际需求取消勾选相应选项，来控制水印在特定操作下的显示情况。
水印样式设置：点击 “水印样式” 旁边的齿轮图标，可对水印的样式进行详细设置。您可以调整水印的不透明度（取值范围 0 - 100%），数值越小水印越透明；还能设置水印的旋转角度（逆时针方向），改变水印的倾斜效果；以及设置水印的重复方式等，以满足个性化的需求。
水印效果查看：点击 “水印效果” 后的 “点击查看” 链接，您可以预览当前设置的水印在文件上的实际显示效果，以便及时调整设置，直到达到满意的效果。
六、注意事项
浏览器兼容性：水印及应用范围管控功能不支持 IE10 及以下版本的浏览器。若使用此类浏览器访问相关页面，可能无法正常显示水印，也无法进行设置操作。建议用户选用符合要求的浏览器，如 Chrome、Firefox 等，以确保功能正常使用。
权限限制：进行水印及应用范围设置操作需要用户具备相应的权限。企业应合理分配权限，确保只有经过授权的人员才能进行相关操作，保障数据的安全性。例如，普通员工可能仅具有查看水印的权限，而管理员才有设置和修改水印及应用范围的权限。
保存设置：在完成水印和应用范围的各项修改操作后，务必点击 “保存” 按钮，使设置真正生效。若未保存设置便直接离开页面，之前所做的所有修改内容将会丢失，只能重新进行设置。因此，及时保存设置是非常重要的操作步骤。
页面链接设置规范：在设置按页面控制的应用范围时，设置页面链接需严格按照提示要求进行操作，仅输入路由地址即可，切勿带上 “?” 之后的参数，以免导致设置错误。
同步操作风险：若涉及应用范围设置的同步操作（如系统更新或数据迁移时），同步操作会直接覆盖目标端已有的设置。因此，在执行同步操作前，请务必谨慎确认，并建议备份重要数据，以免因同步操作导致数据丢失，造成不必要的损失。
报错处理：若在操作过程中遇到 “网页解析失败，可能是不支持的网页类型，请检查网页或稍后重试” 的报错信息，用户需要仔细检查所操作的网页是否符合水印设置规则，或者稍后再次尝试操作，看是否能够解决问题。
文件水印设置说明：水印取值的优先级为：业务模块指定的水印 > 附件存放文件夹关联的水印 > 当前页面配置的水印 。这意味着如果多个地方都设置了水印，会按照此优先级来显示。文件水印配置对图片不生效。即如果您设置了文件水印，在文件中的图片部分不会显示该水印。
七、更多场景示例
1、企业培训资料管理
在企业组织培训活动时，培训资料包含大量内部信息。对于面向全体员工的基础培训资料页面，可将其添加到水印显示的白名单，确保员工知晓资料来源和版权归属；而对于培训讲师的备课资料页面，为了保护备课内容的隐私性，可将讲师的人员范围添加到水印显示的黑名单，同时将备课资料页面添加到按页面控制的黑名单，使讲师在查看备课资料时不显示水印，提高工作效率。
2、项目合作中的数据安全保护
企业与外部合作伙伴进行项目合作时，会涉及大量项目数据的交互。对于合作项目中需要共享给外部合作伙伴的一般性资料页面，可将合作伙伴的 IP 网段添加到水印显示的白名单，同时设置水印内容包含合作项目名称和当前日期，便于追踪数据流向；而对于企业内部的核心项目数据页面，不仅要将外部合作伙伴的人员范围和 IP 网段添加到水印显示的黑名单，还要将这些页面添加到按页面控制的黑名单，防止核心数据泄露。
八、上下游关联功能点
数据追踪与审计系统：水印及应用范围管控功能与企业的数据追踪和审计系统紧密关联。通过精准控制水印的显示范围，企业能够更高效地追踪数据在不同人员、页面或 IP 范围内的流动轨迹，为数据审计工作提供坚实有力的依据。例如，在发生数据泄露事件时，企业可依据水印的显示情况，迅速定位数据泄露的源头，为后续的处理工作争取宝贵时间。
权限管理体系：该功能与企业现有的权限管理体系相互配合、协同工作。水印及应用范围的设置权限，通常与 “水印设置” 权限相关联。同时，在人员范围和 IP 网段的选择过程中，也高度依赖权限管理体系中对人员、部门、岗位以及 IP 访问权限的预先设定。这种紧密的关联确保了只有具备相应权限的人员，才能够进行水印及应用范围的设置和查看操作，进而保障了水印设置的安全性和规范性，有效防止未经授权的人员随意更改设置，对企业数据安全造成威胁。
