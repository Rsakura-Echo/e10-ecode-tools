一、功能背景
在企业日常业务流程中，涉及大量数据录入操作，如员工信息登记、客户资料填写、业务表单提交等。为确保录入数据的准确性和规范性，传统的人工审核方式效率低下且容易出错。正则表达式作为一种强大的文本匹配工具，能有效规范用户输入格式。通过自定义正则校验设置功能，企业可以针对不同的数据类型和业务需求，灵活配置校验规则，自动对用户输入进行实时检查，减少数据错误和不一致性，提高数据质量，为后续的数据处理、分析和业务决策提供可靠基础。
二、功能价值
确保数据准确性：通过设置精确的正则表达式，对输入数据进行严格校验，防止不符合格式要求的数据进入系统，减少数据错误带来的后续处理成本和风险。
提升业务效率：自动化的校验过程取代了人工审核，用户在输入时即可收到错误提示，及时修正，加快数据录入流程，提高整体业务处理效率。
增强系统稳定性：规范的数据输入能减少系统因处理错误数据而引发的异常情况，保障系统的稳定运行，降低维护成本。
适应多样化业务需求：支持自定义设置，企业可根据不同业务场景（如人力资源管理、财务管理、客户关系管理等）的特殊要求，定制专属的校验规则，满足多样化的数据规范需求。
三、功能概述
自定义正则校验设置功能允许用户在后台管理中心配置各类正则表达式校验规则。用户可以创建新的正则表达式，设置其名称、具体表达式内容、校验不通过时的提示语以及显示顺序等参数。这些规则可应用于表单设计器等场景，对用户输入的文本内容进行格式校验，确保输入数据符合预设要求。

五、功能详细说明
1.
进入设置页面
登录后台管理中心，在左侧导航栏中找到 “通用设置” 选项并点击，然后在展开的列表里选择 “自定义正则校验设置”，即可进入自定义正则校验设置的操作页面。页面展示了已有的正则表达式规则列表，包括正则表达式名称、校验不通过时提示语、是否启用以及显示顺序等信息。
2.
新建规则
点击新建按钮：在自定义正则校验设置页面的右上角，点击 “新建” 按钮，打开新建规则页面。
启用开关：新建页面顶部有一个 “启用” 开关，默认处于开启状态。开启后，该正则表达式规则生效；关闭则规则不生效。
正则表达式名称：必填项，在对应的输入框中输入规则名称，用于标识该正则表达式的用途，如 “手机号码校验”“邮箱地址校验” 等，长度限制为 0 - 30 字符。
正则表达式：必填项，可自行输入正则表达式内容，也可在 “常用正则表达式” 列表中选择系统预设的表达式，如 “纯中文”“纯数字”“手机号码” 等。系统预设的表达式能满足常见的数据格式校验需求，若有特殊要求，可自行编写表达式。
校验不通过时提示语：必填项，输入当用户输入不符合正则表达式规则时显示的提示信息，用于指导用户正确输入，长度限制为 0 - 1000 字符。例如，对于手机号码校验规则，可设置提示语为 “格式有误，请输入正确的手机号码” 。
显示顺序：用于设置该规则在规则列表中的显示顺序，数值越小越靠前，默认值为当前规则总数加 1，用户可根据需求修改。
保存规则：填写完所有必填信息后，点击页面右上角的 “保存” 按钮，保存新建的正则表达式规则。
3.
管理现有规则
编辑规则：在自定义正则校验设置页面的规则列表中，点击某条规则对应的编辑图标（通常为铅笔形状），可打开编辑页面。在编辑页面中，可对规则的各项信息（如正则表达式名称、表达式内容、提示语、显示顺序等）进行修改，修改完成后点击 “保存” 按钮使更改生效。
删除规则：在规则列表中，勾选需要删除的规则（可多选），然后点击页面右上角的 “批量删除” 按钮，系统会弹出确认删除的提示框，点击 “确定” 即可删除所选规则。注意，删除规则操作不可逆，请谨慎操作。
4.
在表单设计器中应用规则
在表单设计器中，当添加单行文本等可进行输入校验的组件时，在组件设置区域找到 “自定义正则校验” 选项，点击其设置图标（齿轮形状），弹出自定义正则校验设置窗口。窗口中展示了已创建并启用的正则表达式规则，分为 “常用正则” 和 “表达式” 两部分。“常用正则” 列出了系统预设和用户自定义的常用规则按钮，可直接点击选择；“表达式” 输入框可手动输入正则表达式（若不使用预设规则）。选择或输入正则表达式后，还可在 “校验不通过时提示语” 输入框中输入提示信息（若与后台设置不同），点击 “确定” 完成设置。这样，在用户填写表单时，该组件的输入内容将按照所选正则表达式规则进行校验。
5.
注意事项
正则表达式语法：自行编写正则表达式时，需严格遵循正则表达式语法规则，确保表达式的准确性和有效性。错误的表达式可能导致校验失败或出现意外的校验结果。
提示语准确性：校验不通过时的提示语应简洁明了、准确易懂，能清晰指导用户正确输入，避免使用模糊或容易引起歧义的表述。
规则顺序调整：调整规则显示顺序时，需综合考虑业务逻辑和使用频率等因素，将常用或重要的规则设置在靠前位置，方便在表单设计等场景中快速选择应用。
规则启用状态：确保在表单设计等场景中应用的规则处于启用状态，否则规则不会生效，无法对用户输入进行校验。
六、更多场景示例
1. 人力资源管理场景
在员工入职信息登记表单中，使用自定义正则校验设置功能。对于 “手机号码” 字段，选择 “手机号码” 正则表达式规则，设置校验不通过时提示语为 “请输入正确的 11 位手机号码”，确保录入的手机号码格式正确。对于 “身份证号” 字段，选择相应的身份证号正则表达式规则，规范身份证号输入，提高员工信息录入的准确性。
2. 财务管理场景
在费用报销表单中，对 “金额” 字段设置正则表达式校验规则。选择 “数字” 相关的正则表达式，并设置校验不通过时提示语为 “请输入正确的数字金额”，防止非数字或错误格式的金额数据录入，保证财务数据的准确性和规范性。同时，对于 “发票号码” 等字段，也可根据实际发票号码的格式要求，设置自定义正则表达式进行校验。
七、上下游关联功能点
表单设计器：自定义正则校验设置功能与表单设计器紧密关联。在表单设计过程中，通过表单设计器调用已设置的正则表达式校验规则，对表单组件的输入内容进行实时校验，确保表单数据符合业务要求。表单设计器的组件设置和数据校验逻辑依赖于自定义正则校验设置功能提供的规则支持。
数据存储与处理模块：经过正则校验后的数据，准确规范地存储到系统的数据存储模块中，为后续的数据处理、分析和报表生成等功能提供可靠的数据基础。若数据未经有效校验，可能导致数据存储错误，影响数据处理和分析结果的准确性。
