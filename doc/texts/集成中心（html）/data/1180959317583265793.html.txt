一、功能背景
用户对统一待办中心的全局性功能可以做统一设置，涉及：流程列表显示情况，日志与缓存处理，是否启用消息队列，以及相关的高级设置。
二、功能价值
全局性设置：用户对统一待办中心的全局性功能可以做统一设置。
三、功能说明
功能入口：后台管理中心-集成中心-统一待办中心集成-功能设置；
功能说明：对统一待办中心的全局性功能可以做统一设置。

3.消息队列设置
功能入口：后台管理中心-集成中心-统一待办中心集成-功能设置-消息队列设置；
功能说明：维护消息队列有关的配置，含队列异常的提醒方式和提醒对象；
消息队列的作用：统一待办接口接收数据后，通过校验的合法数据会放入到消息队列中，供其他三方系统去消费；消息队列的数据超过10000个未被消费，消息队列的状态会设置为异常；
注：本功能仅私有化环境支持。
3.1. 启用
功能说明
用于控制是否启用消息队列
控件类型
开关
是否必填
否
默认值
关闭
启用：启用消息队列功能，并显示相应的队列名称及队列状态（正常、异常）；联动显示【队列异常提醒】配置项；
关闭：关闭消息队列功能；
3.2. 队列异常提醒
功能说明
用于控制当消息队列发生异常，是否做相应的提醒操作；
显示条件
【消息队列】启用后，显示本配置项；默认为隐藏；
控件类型
开关
是否必填
否
默认值

启用：启用消息队列异常提醒功能，联动显示【提醒方式】和【提醒对象】这2个配置项，并可对【提醒方式】和【提醒对象】进行配置操作；
关闭：关闭消息队列异常提醒功能；
3.3. 提醒方式
功能说明
用于控制当消息队列发生异常时，使用哪种方式进行提醒操作（下面列出4种提醒方式）；
显示条件
【队列异常提醒】启用后，显示本配置项；默认为隐藏；
控件类型
复选框
复选框选项
提醒到IM、提醒到云桥、提醒到邮箱、提醒到短信；
是否必填
是
默认值
无
3.3.1. 提醒到IM
勾选该选项，相关推送提醒消息到E-Mobile10；
使用E-Mobile10才会显示该提醒项；
3.3.2. 提醒到云桥
勾选该选项，相关提醒信息推送到云桥10对应的系统（例如：企业微信、钉钉）；
使用云桥10才会显示该提醒项；
3.3.3. 提醒到邮箱
勾选该选项，相关提醒信息推送到邮箱（使用系统级邮箱功能）；
并且可配置邮件提醒模板；
使用邮箱功能才会显示该提醒项；
3.3.4. 提醒到短信
勾选该选项，相关提醒信息推送到短信（使用系统级短信功能）；
并且可配置短信提醒模板；
使用短信功能才会显示该提醒项；
3.4. 提醒对象
功能说明
用于控制当消息队列发生异常时，哪些提醒对象可接收到相应提醒消息；

控件类型
复选框
复选框选项
提醒到IM、提醒到云桥、提醒到邮箱、提醒到短信；
是否必填

默认值
无
