1、手动同步：将OA已有组织和人员全量同步至CoreMail邮箱
2、实时同步：

3、门户元素支持展示CoreMail企业邮箱未读邮件列表与未读邮件数量（无测试环境，暂无邮件展示图片）
门户导航栏图标、门户导航栏角标支持coremail企业邮箱查看未读邮件数量

二、
功能使用说明
场景案例
实现将OA已有组织和人员全量同步至CoreMail邮箱，同时OA中组织人员变更，实时同步到CoreMail邮箱，门户元素可查看未读邮件和数量。
操作步骤
1.Coremail邮箱服务配置
IP白名单配置：系统设置-常用设置-API 配置允许OA访问ip白名单
OA服务ip地址配置为白名单：
2.集成中心-邮箱集成
点击CoreMail、CoreMail-webservice邮箱新建
除可以在【集成中心】-【邮箱集成】直接新建；也可以到ESB中心连接器中新建配置，ESB连接器中只是提供一个入口，功能都需要在【集成中心】-【邮箱集成】查看
配置项
说明
启用
默认开启，控制整个功能是否可用的开关
名称
任意
产品类型
CoreMail邮箱/CoreMail-webservice邮箱
所属机构
开启分权时显示
绑定邮箱字段
以此字段的值作为邮箱系统的账号
启用同步
默认开启，开启后可将OA组织和人员同步至邮箱系统
关联单点登录标识
关联后可邮箱门户元素查看未读邮件、门户导航栏可点击登录邮箱系统
provider id
默认填写1即可
域名
CoreMail邮箱提供，例如人员邮箱为test01@weaver.com.cn，填写weaver.com.cn即可
邮箱地址
填写coremail邮箱域名地址或者ip地址，例如：192.168.45.26或者e-cology.com.cn
邮箱端口
默认6195，如coremail邮箱有变更，根据实际修改
组织ID
CoreMail邮箱提供，将OA组织同步到此组织下，coremail可以新建多个组织标识
注：coremail邮箱参数获取参考：组织用户-组织管理 中可查看域名和组织id
3.维护个人信息卡片邮箱
配置示例“绑定邮箱字段”设置为“邮箱”
人力卡片邮箱字段维护正确的CoreMail邮箱值
4.手动同步
按照实际需要选择是否配置此步骤。
【集成中心】-【邮箱集成】打开新建的配置可点击同步，点击后将所选范围下的组织和人员同步至CoreMail邮箱。
5.门户邮箱集成元素
涉及权限项：【门户】
按照实际需要选择是否配置此步骤
【后端管理中心】-【门户】-【设计】
搜索“邮箱”添加
内容来源选择集成中心-邮箱集成新建的配置，此处也可按需配置邮件列表、提示信息等门户元素样式
效果：若配置绑定了集成登录标识，点击右键则可登录Coremail邮箱
6.门户导航栏
涉及权限项：【界面配置中心】
按照实际需要选择是否配置此步骤
【后台管理中心】-【界面配置中心】-【导航设置】-【PC网页版导航或PC客户端导航】，启用“CoreMail邮箱”
效果：点击可单点登录至邮箱，并可查看未读邮件的数量
