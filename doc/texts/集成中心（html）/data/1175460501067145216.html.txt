一、功能介绍
注意：需要购买统一认证-集成登录功能。
实现OA系统的单点登录功能，以便无缝接入用友NC。
二、功能概述以及使用说明
注意：e10支持三种用友NC套件——用友NC Cloud、用友NC6.X和用友NC5.X。请根据客户现有的用友NC版本，选择相应的套件进行配置。
1、用友NC操作
为了配置用友NC套件，需要用友NC提供相应的单点登录参数值。
1.1、用友NC Cloud
根据用户NC Cloud的单点接口文档，需要提供数据源名称、第三方系统id和第三方密钥。
1.2、用友NC6.X
根据用户NC60的单点接口文档，需要提供账套编码。
1.3、用友NC5.X
根据用户NC5.X的单点接口文档，需要提供账套编码。
3、功能入口
配置新建集成登录用友NC系统提供了两个功能相同的入口，推荐使用入口1：
入口1：
【集成登录】-【集成登录设置】-【用友NC】
需要集成登录权限：
入口2：
此入口需购买ESB中心非标
【ESB中心】-【连接器】-【财税管理】-【用友NC】-【套件】-【集成登录】
需要ESB-连接器权限：
4、配置说明
4.1、用友NC Cloud套件
4.1.1、选择套件快速创建
通过入口1和入口2，用户可以迅速设置用友NC Cloud 的集成登录功能。除了必须手动配置的参数外，其他设置将默认生成，并且这些默认参数可以在详细设置中进行修改。

4.4、共享范围
需要在共享范围内设置，哪些人员能够通过集成登录实现对用友NC的单点登录。
4.5、门户集成登录组件
按照实际需要选择是否配置
【后端管理中心】-【门户】-【设计】
模块组件中搜索“集成登录”可添加集成登录组件。
设置后，有权限人员在门户中可展示集成登录组件，点击应用可直接单点登录用友NC。
4.6、构建应用
按照实际需要选择是否配置。
【后端管理中心】-【应用管理】-【构建应用】
可配置菜单，菜单中集成登录可选择已配置的集成登录。
可配置单页面内容， URL地址配置集成登录的单点登录地址。
集成登录的单点登录地址可通过集成登录里面【更多】-【单点登录地址】进行复制。
最终效果，有权限人员在应用中可点击应用单点登录到用友NC。
