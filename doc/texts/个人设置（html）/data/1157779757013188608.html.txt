绑定手机、邮箱后，可用手机号、邮箱登陆同一个账号，方便您的管理。
邮箱绑定
1.
点击左下角设置-设置我的信息，打开用户资料页面
2.
在邮箱的后面有个绑定链接，点链接弹出邮箱绑定对话框
3.
输入邮箱后，点击发送验证码按钮，系统会发送一封包含验证码的邮件到您刚刚输入的邮箱中
4.
在验证码输入框输入获取的验证码，点击确定，就完成了邮箱的绑定
手机绑定
1.
点击左下角设置-设置我的信息，打开用户资料页面
2.
在手机项的后面有个绑定链接，点链接弹出手机绑定对话框
3.
输入自己的手机号后，点击发送验证码按钮，系统会以短信的方式将验证码发送到您刚刚输入的手机上
4.
在验证码输入框输入获取的验证码，点击确定，就完成了手机号的绑定
