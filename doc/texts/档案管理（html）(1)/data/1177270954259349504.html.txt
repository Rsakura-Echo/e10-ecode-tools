1.
档案 archives
国家机构、社会组织或个人在社会活动中直接形成的有价值的各种形式的历史记录。
2.
电子档案 electronic records
通过计算机等电子设备形成、办理、传输、存储的，具有保存价值并归档保存的各种信息记录。
3.
文书档案 administrative archives
反映党务、行政管理等活动的档案。
4.
科学技术档案 scientific and technical archives
反映科学技术研究、生产、基本建设等活动的档案
5.
电子会计档案 electronic accounting archive
在会计核算工作中由电子计算机直接形成或接收、传输、存储并归档,记录和反映单位经济业务事项，具有凭证、查考和保存价值的电子会计资料
6.
全宗 fonds
一个国家机构、社会组织或个人形成的具有有机联系的档案整体。
7.
全宗号 fonds number
档案所属全宗的代码
8.
档号 reference code
以字符形式赋予档案的一组唯一代码,用于反映、固定和识别档案排列顺序来源:DA/T 1-2000,]
9.
类别号 series number
按照分类方案赋予全宗内各层级档案类别的代码。
10.
档案门类代码 archival category code
档案所属门类的代码。
注:档案门类代码是一级类别号,如文书档案代码、科技档案代码等.
11.
案卷号 file number
案卷排列的顺序代码
注:按组或册整理的档案可用组号或册号
12.
件号 item number
归档的文件排列的顺序代码
13.
页号 page number
案卷内文件每页排列的顺序代码
14.
归档 filing
办理完毕且具有保存价值的文件经系统整理交档案室或档案馆保存的过程。
15.
接收 accession
档案馆、档案室按照国家规定收存档案的过程。
16.
鉴定 appraisal
判定档案真伪和价值的过程。
17.
保管期限 retention period
对档案(2.1)划定的存留年限。
18.
保管期限表 records retention schedule
规定档案保管期限的文件。
19.
销毁 destruction
经过鉴定对失去价值的档案作毁灭性处置的过程。
20.
销毁清册 destruction list
登录被销毁档案题名、数量等内容并由责任人签署的文件。
