三、常见问题
1.接口调用失败的常见原因
向薪人薪事服务器发送 API 请求时，必须在请求头里带上access_token，方便服务器认证用户身份。
获取 API Token 的方式参考下方链接：https://api.xinrenxinshi.com/doc/v3/page/guide/authentication.html
