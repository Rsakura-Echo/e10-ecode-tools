一、功能背景及价值
本章主要介绍如何发起一条报销流程，基于报销流程可配置费用标准、预算、分摊等多种费控规则，并且可以对接银企互联模块，实现线上转账。电子化报销极大的提升了员工、财务的效率，也让企业对成本和费用实现更精准、合理的把控
二、功能详细说明
1、方式一
首页 -> 【快速发起】-> 点击需要提交的工作流 -> 进入填写流程表单页面
2、方式二
点击【首页】菜单 -> 点击【新建申请】 -> 进入新建申请页面 -> 点击需要提交的工作流 -> 进入填写流程表单页面
3、方式三
点击【我的申请】/【待我审批】/【申请查询】菜单-> 点击【新建申请】-> 进入新建申请页面 -> 点击需要提交的工作流 -> 进入填写流程表单页面
三、上下游关联功能点
提交报销流程前，需要先在流程模块新建工作流 ，然后在电子费控模块关联工作流
四、注意点
在电子费控模块发起的流程，也可以在流程模块查看。但是在流程模块发起的流程，不可在电子费控流程列表查看
