一、
功能背景
指标看板是数据分析中一种重要的指标类可视化组件，其核心目的是一目了然地展示关键数据或业绩状况。通过将核心指标以醒目的卡片形式呈现，用户可以快速掌握业务现状、发现问题并及时制定应对策略。
二、
功能价值
1.
通过关键指标（如销售额、用户留存率、转化率）实时追踪业务表现，快速发现异常波动。
2.
将主观经验转化为客观数据，避免“拍脑袋”决策。
3.
标准化指标定义，避免部门间数据分歧。
4.
将指标与业务目标绑定，形成“目标设定→执行→监控→优化”的闭环。

2.
数据预览
在指标分析页面增加分析维度字段，然后数据预览tab菜单栏选择分析的维度字段，可以选择柱状图，曲线图等方式展示
3.
指标授权
可以将创建好的指标共享给他人，可以分配查看权限（仅可查看该指标数据）、维护权限（可更新修改该指标的配置信息）。
4.
手动更新
手动更新指的是立马触发该指标数据重新计算，在指标的配置页面可以设置定时更新计算指标的时间和频率，如果需要当前的指标信息，则可以点击手动更新。
5.
复制
点击复制按钮，会在右侧出现和原指标相同配置的页面，名称默认为原指标+复制，点击保存即可新建一份相同指标
6.
移动到
点击移动到可以将该指标移动到别的分组下面
7.
指标迁出
点击指标迁出会生成一个本地文件，然后在指标新建页面点击指标本地迁入即可将该指标迁移到别的环境中去
8.
删除
可以将已经建好的指标删除
