一、功能背景及价值
员工开具企业发票时会存在抬头错误的情况，企业可设置白名单，当购买方和白名单信息相同时才允许录入，杜绝错票
二、权限说明
系统管理员角色默认有此功能权限，其他角色需要授权
三、功能详细说明
1、新建白名单
方法1：点击右上角【新建】按钮 -> 填写白名单信息 -> 点击【确定】按钮
（1）可根据发票类型设置校验内容，如增值税发票校验抬头&税号&开户行&地址电话
（2）可设置是否忽略特殊字符，如开户行地址电话中填写了“-” ，可忽略符号不校验
（3）可设置白名单生效、结束时间
（4）可设置白名单适用范围，范围内的用户录入发票时受白名单规则控制
方法2：点击右上角【excel导入】按钮 -> 下载模板 -> 导入excel
四、上下游关联功能点
无
五、注意点
1、默认校验发票购买方信息是否和白名单一致
2、可选择购买方或销售方，此时发票购方或销方只要有一处校验内容和白名单相同就允许录入
