# 24 - ecode 平台版本历史与浏览器支持

## 概述

本文档记录 ecode 开发平台各基线版本的新增/优化功能，以及支持的浏览器版本。

---

## 一、平台新功能介绍（1.5）

部分功能发布日志如下，完整功能说明请查阅：E10 e-code开发平台功能说明文档

| 基线版本 | 新增/优化功能 | 说明 |
|---------|-------------|------|
| 10.0.2509.01 | API 准入控制 | ecode 新增针对前端 request 和 axios 请求拦截的 api 准入控制 |
| 10.0.2508.01 | JAR 包审核 | ecode 二开 jar 包新增审核功能 |
| 10.0.2507.01 | 代码屏蔽调试 | ecode 支持按应用&资源文件路径屏蔽调试；自动识别报错并提示 |
| 10.0.2505.01 | 分权管理 | ecode 项目应用支持分权管理 |
| 10.0.2504.01 | 全局代码搜索 | ecode 支持全局代码搜索 |
| 10.0.2503.01 | 特色主题 | ecode 支持特色主题项目类型 |
| 10.0.2502.01 | 代码屏蔽 + Vue 支持 | ecode 支持快捷屏蔽代码块调试功能；支持 Vue 开发 |
| 10.0.2412.01 | 智能化开发 | ecode 支持智能化开发 |
| 10.0.2411.01 | JAR 包管理 | ecode 监控平台 jar 包部署与维护 |

## 四、数据库操作接口演进

| 基线区间 | 接口 | 特点 |
|---------|------|------|
| 241201 及以前 | `DataSetService.executeSql()` | 按 SourceType + groupId 执行 SQL |
| 250101-250401 | `/sapi/secondev/ds/executeSql` | 通过 SAPI 调用，LOGIC 不支持预编译 |
| 1201 及以上 | `/sapi/secondev/ds/executeSqlAll` | LOGIC/EXTERNAL 统一接口，支持预编译+事务 |
| 250501 及以上 | `SecondevDataSetUtil` | 封装工具类，直接注入使用 |
