# 23 - 自定义浏览按钮

## 概述

E10 支持配置自定义浏览按钮，用于在表单或页面中弹出选择器（如选择人员、选择部门、选择数据等）。

参考文档：
- [E10 自定义浏览按钮配置指南](相关链接)
- [E10 浏览按钮 & 自定义浏览框 & 浏览按钮数据范围](相关链接)

---

## 组件参数说明

在使用自定义浏览按钮组件时，核心参数如下：

| 参数 | 说明 |
|------|------|
| `module` | 自定义浏览框统一为 **`bcw`** |
| `type` | 固定前缀 **`customBrowser_`** + 自定义浏览框的标识 |

### 标识获取方式

后台通用设置 → 自定义浏览框配置 → 浏览框基本信息 → **标识**

---

## 配置流程

1. **后台配置**：进入后台通用设置 → 自定义浏览框配置，创建浏览框并获取标识
2. **前端使用**：在表单或页面中使用浏览按钮组件，设置 `module="bcw"` 和 `type="customBrowser_标识"`
3. **数据范围**：可在浏览框配置中设置数据筛选条件

---

## 示例

```jsx
// 在表单字段中使用自定义浏览按钮
<BrowserButton
  module="bcw"
  type="customBrowser_myCustomSelector"
  // 其他 props...
/>
```
